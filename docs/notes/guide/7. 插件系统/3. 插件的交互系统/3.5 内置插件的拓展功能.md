## 内置插件表

- `UnifiedRegistry`: 统一命令注册器，提供命令和过滤器注册功能。
- `SystemManager`: 系统管理插件，提供插件管理、运行状态管理等功能。

## UnifiedRegistry 拓展功能

### on_request 注册器

为插件方法或普通方法注册 Request 事件处理器。
